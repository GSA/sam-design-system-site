Select

Select elements allow users to pick one (or more) items from a list of options.

Deviation from the US Web Design Standards

The SAM Web Design Standards combines dropdowns, checkboxes, and radio buttons under the genral heading &lquo;Select&rquo; for the purposes of the UI kit.
