Select

Select elements allow users to pick one (or more) items from a list of options.

See US Web Design Standards for details regarding the following elements:

  • Dropdown
  • Checkboxes
  • Radio buttons

Deviation from the US Web Design Standards

The SAM Web Design Standards combines dropdowns, checkboxes, and radio buttons under the genral heading &lquo;Select&rquo; for the purposes of the UI kit.

    Select

    Required keys
    • type: dropdown|checkbox|radio
    • options: An array of key-value pairs where the "key" is the option POST/GET value and "value" is the label for the option.
    Optional keys
    • selected: An array of option values to mark as selected/checked. Note: Only the checkbox type allows multiple selections.
    • disabled: An array of option value to mark as disabled. Note: Only the checkbox type allows multiple selections.
