SAM.gov Design System

This is a federal open source project under active development. It is being designed for use in award or other federal financial systems. It provides public access and encourages sharing of federal code across the government (Federal Policy About Sharing Source Code).

Get Started

Welcome to the SAM.gov Design System Site. This design system is in an MVP state and under active development. Please come back frequently to see our progress!
