<h1>Complex Data Entry Page</h1>
<p>A simple data entry page has the following characteristics.</p>
<ul>
<li>The page has a form where the user enters and submits information</li>
<li>The user typically enters information in multiple sessions (&gt; 30 min). The user may need to save as a draft to gather further information, or there may be workflow steps involving multiple users.</li>
<li>The form has multiple conditional statements that affect what information the user must provide</li>
<li>The form should be broken into distinct sections due to its length or complexity</li>
<li>The form has multiple fields that require a complex editor</li>
</ul>

<h2>Toolbar</h2>
<p>The toolbar on a complex data entry page has a toolbar with the following functions.</p>
<ul>
<li>Cancel</li>
<li>Next (optional)</li>
<li>Previous (optional)</li>
<li>Done</li>
<li>Show Errors (optional)</li>
</ul>
