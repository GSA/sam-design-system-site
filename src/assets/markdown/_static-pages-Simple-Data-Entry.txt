<h1>Simple Data Entry Page</h1>
<p>A simple data entry page has the following characteristics.</p>
<ul>
<li>The page has a form where the user enters and submits information</li>
<li>The user typically enters all information in one sitting (&lt; 30 min) with little preparation</li>
<li>The form has few or no conditional statements that affect what information the user must provide</li>
<li>The form has no need to be broken into distinct sections due to its length or complexity</li>
<li>The form has few or no fields that require a complex editor</li>
</ul>

<p>The toolbar on a simple data entry page has a toolbar with the following buttons.</p>
<ul>
<li>Cancel</li>
<li>Done</li>
</ul>
