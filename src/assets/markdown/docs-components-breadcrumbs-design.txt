<h2>Search Breadcrumbs</h2>
<p><em>Use with award data and help searches</em></p>
<ul>
<li>The search results page does not have breadcrumbs.</li>
<li>Display pages accessed via search have a single &quot;Back to Search Results&quot; breadcrumb.</li>
</ul>
<h2>&quot;File System&quot; Breadcrumbs</h2>
<p><em>Use with the &quot;file system&quot; available for data services</em></p>
<ul>
<li>The breadcrumbs mirror the file system as the user is drilling down into it</li>
</ul>
<h2>Other Breadcrumbs</h2>
<p><em>Use with workspace, reports, profile, and help pages</em>
Other breadcrumbs following the information architecture of the site, orienting users to where the page lies with the site</p>
<h3>Examples:</h3>
<ul>
<li>Workspace and Help Landing Pages: None</li>
<li>Role Management Workspace Page:  <strong><em>Workspace / Role Management</em></strong></li>
<li>Role Management Grant Access Page: <strong><em>Workspace / Role Management / Grant Access</em></strong></li>
</ul>
